# LDPC-Iterative-Bit-Flipping-family-decoders
Iterative Bit-flipping LDCP decoders: Implementation-efficient Reliability Ratio Based Weighted Bit-Flipping (IRRWBF) Decoding for LDPC codes, LDPC Weighted Bit Flipping Algorithm (WBF), Modified Weighted Bit Flipping Algorithm (IWBF), Weighted Bit Flipping Algorithm (WBF).
